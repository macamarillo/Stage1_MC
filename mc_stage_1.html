Lesson #1: The Basics of the Web and HTML

Overview of the Web

The Web is composed of multiple HTML (Hypertext Markup Language) pages. The major pieces of the Web are you, your computer and browser, HTTP (protocol that browsers use to connect to servers), the internet (which uses HTTP to communicate with you), your browser, and servers.

HTML

HTML, or HyperText Markup Language, is the heart of the Web and is made up of text/content (what you see), markup tags (what it looks like), references to other documents (e.g., images and videos, and links to other pages, such as Google.com.
HTML documents have many components, but what was focused on in this lesson was primarily the body/contents of an HTML document. In the lesson, I was introduced to a number of tags and learned the difference between inline and block elements. Block elements form an invisible box around text that you can then, for example, assign a height and width to. I thought this was pretty cool.

What Computers Know

Computers are often called stupid. What this means is that a computer needs very specific instructions from a programmer in order to know how to perform. The computer only knows how to do exactly what you tell it to do.

Good news--programmers can't remember everything and aren't expected to!

This was the most encouraging part of the lesson, i.e., programmers can't remember everything. It was good to know that there are good resources available to help you remember commond HTML syntax.

Lesson #2: Creating a Structured Document with HTML Subtitles

Focus of Course

This course focused on showing a methodical way of breaking down a current web page and recreating it for your own purposes.

The Way Websites Work

A website can be thought of like your house.
  • HTML (HyperText Markup Language) files are like the structure, where the walls are placed and is the standard markup language used to create web pages.
  • CSS (Cascading Style Sheets) files are then the style of your house. Style sheet language is used for describing the look and format of a dcoument written in markup language.
  • JavaScript files are the interactive components of your house.

Web Page Structure

HTML is the language of the Web and it has specific syntax and rules. The basic word in the HTML language is a tab. Browsers turn HTML tags into element that form a tree-like structure.

Boxes

Everything on the Web is a box. So, when you receive an assignment to turn a page of content to a Web page, you must "draw" a visual box around every element you see. Boxes on a page can be different sizes and in different locations. They can also be marked so that style can be applied to them later. The best way to do that is to give each element a class atrribute.

Class attributes

  • Can be thought of as labels on boxes, i.e., something that reminds you of what's inside the box.
  • Boxes can have several labels or attributes applied to them. The same applies to HTML tags; they can belong to multiple classes.
  • It's a good idea to have class names that make sense so that you understand the purpose of the "box."

Lesson #3: Adding CSS Style to HTML Structure Subtitles

Focus of Course

This course focused on CSS, code written to control the "style" of HTML elements, and on how to avoid repetition by using CSS. Avoiding repetition helps programmers avoid rewriting the same code in multiple places, which can lead to errors. It also helps a programmer achieve consistency in style.

Inheritance

Inheritance is key in CSS. It relies on the "document tree" referred to earlier. Descendant elements can inherit CSS property values from any ancestor element that encloses them. Inheritable properties include color, font, letter-spacing, line-height, list-style, text-align, text-indent, text-transform, visibility, white-space and word-spacing. Non-inheritable properties include background, border, display, float and clear, height and width, margin, min-mx

CSS Selectors

Selectors are what allows you to target specific HTML elements and apply style to them. A good reference for CSS selectors was provided.

How to Include CSS Styling on Your Web Page

  • Method 1: Write CSS in the "head" tag of your HTML.
  • Method 2: Link your HTML to a separate CSS file.
  • Method 3: Write your style inline with your HTML.

Element Sizes

The size of your actual element is equal to the sumy of the border, padding, and content width. The sample code shown below makes the calculation of the element size include both the border and padding so that it's easier to make layouts.
